PathMap Dictionary Configuration

Most of Jetty's configuration uses the com.mortbay.HTTP.PathMap class. PathMap is an implementation of Dictionary that uses a best match of a path prefix to select an Object from the dictionary. Thus an understanding of PathMap is required to configure Jeeves.

PathMap will always return the longest matching path prefix that it contains. Paths may be matched exactly, as an arbitrary prefix or as a path prefix.
