Jetty 3.x.x Releases

Jetty 3 is currently in BETA release and is a significant revision of the Jetty 2 code base. The aims of the Jetty 3 developement include:
  • Fully RFC2616 compliant implementation of HTTP/1.1.
  • Implement the 2.2 Servlet API.
  • Remain efficient, embedable & extensible.
  • Improved and simplified configuration.
  • Utilize java collection classes for improved performance.
  • Improved coding standards and increasing test coverage.
