Welcome to Jetty Wiki
Jetty is an OpenSource  HTTP Servlet Server written in 100% Java. It is designed to be light weight, high performance, embeddable, extensible and flexible. This is the Wiki documentation site for the Jetty 3 server. The idea of a Wiki server for the documentation is that ANYBODY can edit these pages in order to:
  • Add new content.
  • Clarify existing content.
  • Ask in-line questions for others to clarify.
  • Fix the many spelling mistakes entered by GregWilkins
  • Wreak havoc deleting and destroying content.
