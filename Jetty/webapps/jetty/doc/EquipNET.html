The Mort Bay Jetty server is being used as the embedded web server for the EquipNET eQonnector.

The eQonnector is a "Set Top Box" for semiconductor process tools. It acquires information from the tool's SECS interface, or directly from sensors on the tool, and publishes information as web pages.

The eQonnector is marketed by PRI Automation. For more information see www.pria.com/products/fms/eqonnector/pr_fms_eqonnector_index.htm.
