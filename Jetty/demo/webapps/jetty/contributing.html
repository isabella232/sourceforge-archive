Comments, contributions, feedback, bugs, testing, sponsorship etc. are all welcome, please....

Jetty Discuss

This is the mailing list where much of the public discussion regarding development issues occurs. If you are interested in contributing to the Jetty project, this is a must read. 

        www.egroups.com/group/jetty-discuss/
Jetty Tasks

The official list of subprojects and tasks is kept on Source Forge. 

        sourceforge.net/pm/?group_id=7322


The unofficial task list can be found at JettyWishList.

Jetty CVS (Concurrent Versions System)

As open source software, a developer can download, modify, and contribute to the active code base of Jetty server. The Jetty code base resides in a CVS repository hosted by Source Forge.

To learn more about CVS, check out www.cvshome.org/new_users.html .

To go directly to the Jetty CVS page on Source Forge, click sourceforge.net/cvs/?group_id=7322 .

For quick instructions on using CVS to download the Jetty source:
