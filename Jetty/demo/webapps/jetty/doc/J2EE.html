Can Jetty be integrated with JBoss or other J2EE?
Yes!

Jetty is-being/has-been integrated with JBoss to run all in the same VM for a superfast package which includes most of the J2EE technologies (and all the really important ones!). Check the current status on the jboss site ( www.jboss.org ).

For other J2EE servers, the answer is Yes it could be, but probably isn't yet.
