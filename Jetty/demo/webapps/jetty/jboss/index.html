Jetty Jboss Integration

Jetty has been integrated with the JBoss Application server. Since JBoss 3.0 it has been shipped as the default web tier and is called JBossWeb.

The latest Jetty for JBoss is also available via the Jetty download pages and Jetty.sar can replace JBossWeb.sar to get the latest greatest version.

For more information about Jetty in JBoss, see the Jetty JBoss FAQ.
