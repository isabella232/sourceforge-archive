SECTION: 950-JSP TITLE: JSPJikes QUESTION: Can I use Jikes to compile JSPs? Jasper, the JSP compiler now uses ant to compile the JSPs and can thus use any compiler that ant is capable of using. The initParameter 
compiler
can be set on the JSPServlet to specify which ant compiler option to use. This can be set in webdefault.xml for all web applications or in jetty-web.xml for a specific web application.
