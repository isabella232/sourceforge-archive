Jetty is an Open Source HTTP Servlet Server written in 100% Java. It is designed to be light weight, high performance, embeddable, extensible and flexible, thus making it an ideal platform for serving dynamic HTTP requests from any Java application.

This is an ALPHA release of Jetty 3, which is a significant revision of the Jetty 2 code base. The aims of the Jetty 3 developement include:

  • Fully RFC2616 compliant implementation of HTTP/1.1.
  • Decouple HTTP server from specific Servlet container.
  • Integrate with the Jakarta Servlet 2.2 Container.
  • Integrate with the Jetty Servlet 2.1 Container.
  • Remain efficient, embedable & extensible.
  • Utilize java collection classes for improved performance.
  • Improved coding standards and increasing test coverage.
